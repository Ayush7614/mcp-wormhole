# Integration logos

Bundled SVG brand assets for the docs site integration grid.

Sources:
- [SVGL](https://svgl.app) (MIT) — Cursor, VS Code, OpenAI, Windsurf, Google, LangChain, etc.
- Official favicons / brand assets — LlamaIndex, Mastra, OpenClaw
- Simple Icons (CC0) — CrewAI, Moonshot AI, MCP

Logos are stored locally so GitHub Pages does not depend on external CDNs.

To refresh a logo:

```bash
curl -fsSL "https://svgl.app/library/cursor_light.svg" -o site/public/logos/cursor.svg
```
